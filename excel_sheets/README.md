# Waves Beauty Salon — Profit Analysis (Google Sheets Project)

> **Note:** The project itself (tables, calculations, and comments) is written **in Russian**. This description is provided **in English**.

## Project Overview

This project analyzes the financial performance of a small beauty salon specializing in curly hair — **Waves**. The salon has been operating for about **six months** and wants to identify opportunities to **increase profit**.

All business data is recorded manually in **Google Sheets**, and the dataset provided by the client contains operational and financial records exported from these spreadsheets. The entire analysis — including data cleaning, calculations, and summaries — was performed **exclusively in Google Sheets**.

## Objectives

The analysis focuses on two main areas:

**1. Promotion Campaign Analysis**

- Identify which **promo codes and campaigns** are used most often  
- Evaluate how promotions affect **customer acquisition**  
- Estimate how much **revenue is lost due to discounts**  
- Calculate **average order value (AOV)** for different promo campaigns  

Clients can receive discounts of **5%, 10%, 15%, 20%, or 50%**.

**2. Products and Services Analysis**

- Determine which **services and products** are most popular  
- Identify key **revenue drivers**  
- Understand customer demand patterns
